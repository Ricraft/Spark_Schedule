# Data Directory

This directory contains user data files that are automatically created by the application.

## Files

- `courses.json` - Course information
- `course_groups.json` - Course grouping data
- `schedule_data.json` - Schedule configuration
- `tasks.json` - Task and todo items
- `gpa_records.json` - GPA records
- `daily_notes.json` - Daily notes
- `settings.json` - Application settings
- `shici_cache.json` - Poetry cache
- `jinrishici_token.txt` - Poetry API token

## Note

These files contain personal data and are excluded from version control.
The application will create them automatically on first run with default values.
