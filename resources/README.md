# 资源文件说明

此目录用于存放应用程序的图标和其他资源文件。

## 图标文件

### 应用图标
- `icon.ico` 或 `icon.png` - 应用程序主图标（显示在窗口标题栏和任务栏）
- 推荐尺寸：256x256 或更大

### 托盘图标
- `tray_icon.ico` 或 `tray_icon.png` - 系统托盘图标
- 推荐尺寸：16x16 或 32x32（Windows 系统托盘标准尺寸）
- 如果不存在，将使用应用图标作为托盘图标

## 图标制作建议

1. 使用简洁的设计，确保在小尺寸下清晰可辨
2. 托盘图标应该使用高对比度，便于在不同主题下识别
3. 支持透明背景的 PNG 格式
4. Windows 推荐使用 ICO 格式（包含多种尺寸）

## 如何添加图标

1. 将图标文件放入此目录
2. 确保文件名为 `icon.ico`/`icon.png` 或 `tray_icon.ico`/`tray_icon.png`
3. 重启应用程序即可生效
