# ✨ Spark Schedule v2.3.0

> 点燃你的学习热情，让每一天都闪耀光芒

## 📦 下载

**Windows 用户**：下载 `SparkSchedule-v2.3.0-Windows.zip`，解压后运行 `SparkSchedule.exe` 即可使用。

**文件大小**：约 284 MB

## 🌟 版本亮点

### 极致美学设计
- ✨ Modern UI 设计语言，沉浸式视觉体验
- 🎨 支持亚克力半透明磨砂质感
- 🌈 自定义背景图片和透明度调节
- 🖼️ 三种表头风格：默认实色 / 半透明 / 全透明

### 智能高效导入
- 📥 支持 HTML、Excel、文本文件一键导入
- 🔄 自动识别单双周课程
- 📍 智能解析教室和教师信息
- ⚡ 告别繁琐的手动录入

### 深度个性定制
- 📅 自定义每日节数和作息时间
- 🎯 灵活的单双周设置
- 🎨 为每门课程设置专属颜色
- ⚙️ 完美适配各类校园作息

### 贴心课程助理
- 🔔 课前桌面通知提醒
- 🌡️ 实时天气信息（和风天气 API）
- 📖 每日诗词推荐（今日诗词 API）
- 💾 系统托盘常驻，随时唤醒

### 学习数据管理
- ✅ 任务待办清单
- 📊 GPA 成绩统计
- 📈 学分追踪管理
- 📝 每日笔记记录

## 🚀 快速开始

1. **下载并解压** - 下载 `SparkSchedule-v2.3.0-Windows.zip` 并解压到任意目录
2. **运行程序** - 双击 `SparkSchedule.exe` 启动应用
3. **导入课程** - 从教务系统导出课表，使用导入功能一键添加
4. **个性设置** - 在设置中调整外观、作息时间等
5. **开始使用** - 享受高效便捷的课程管理体验

## 📋 系统要求

- **操作系统**：Windows 10/11 (64-bit)
- **内存**：建议 4GB 及以上
- **磁盘空间**：约 500MB
- **网络**：可选（用于天气和诗词功能）

## 🔧 技术栈

- Python 3.8+
- PyQt6 6.6+
- 和风天气 API
- 今日诗词 API
